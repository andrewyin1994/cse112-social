Our Ten Findings

  1. h1-h6 has priority associated with it. Inside section and article tags, h1 becomes the same size as h2. This is because h1 should be used in the header tag of the document.
  2. Articles are indenpendent item section of content. Aside is tangenitally related information.
  3. A section is a generic section of a document or application. A section is a thematic grouping of content, typically with a heading.
  4. The hr tag is the horizonal rule. It represents a paragraph-level thematic break. It basically creates a line through the page.
  5. itemscope is used to help syndicate data. Syndication is when you have another website's content on your website.
  6. With table tag, if you put a paragraph tag above the caption block, you get an error (even though it displays).
  7. The pre tag considers formatting. It is used for preformatted text and it doesn't ignore whitespace.
  8. The param tag is used to define parameters for plugins embedded with an object element.
  9. The main tag is used for the main content of the body of the document and it includes content that is unique to that document.
  10. Authors must not include more than one main element in a document. Also, authors must not include the main element as a descendant of an article, aside, footer, header or nav element.
